## Setup

### Main folder:

Copy the file 'config.private.json.example' to new file 'config.private.json'.
Edit 'config.private.json' to have your mysql database details.
Copy 'config.private.json' to 'config.public.json' and remove the database password.

### Frontend folder:

Run 'npm install'

### Backend folder:

Run 'npm install'.
Run 'node create_database.js'.

## Run

### Backend folder:

Run 'npm start'

### Frontend folder:

Run 'npm run dev' when testing
Run 'npm run prod' when in production
